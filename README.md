# Cloud Computing Labs 3 and 4

Student: Jiang Chengjun

This repository records Lab 3 and Lab 4 for the Introduction to Cloud Computing course.

## Lab 3 - Kubernetes and Minikube

The workflow follows the Kubernetes tutorials:

- Hello Minikube
- Deploy an app
- Explore the app
- Expose the app
- Scale the app
- Update the app

## Lab 4 - Istio Getting Started

The workflow follows the Istio getting started guide:

- Install Istio
- Install Gateway API CRDs
- Deploy Bookinfo
- Configure the gateway
- Access the application
- Install and inspect dashboard add-ons

## Evidence

The GitHub Actions workflow `.github/workflows/lab3-lab4.yml` runs the experiments on an Ubuntu runner and uploads logs under the `evidence` artifact.
